# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [1.0.0] - 2026-01-04

### Declaration

**v1.0.0 marks the beginning of semantically versioned contracts, not functional stability.**  
This version establishes the baseline for SemVer compliance going forward.  
Breaking changes, additions, and fixes will be tracked according to SemVer rules from this point.

**IMPORTANT**: profile-ui releases are immediately published to GitHub Packages. Release tag = deploy.

### Added

- Design system components with B&W aesthetic
- Tech area variants support
- Component primitives (Button, Input, Card, etc.)
- Layout components
- React hooks for common patterns
- Typography tokens and utilities
- Dual package export (ESM + CJS)
- Comprehensive exports for subpaths (tokens, styles, primitives, components, layouts, hooks, utils)

### Infrastructure

- tsup build system
- GitHub Packages publishing
- TypeScript declarations
- CSS globals export
